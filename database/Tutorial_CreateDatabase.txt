Hướng dẫn tạo database cho project SchoolManager

1. Cấu hình kết nối trong application.properties
   - Mở file src/main/resources/application.properties
   - Kiểm tra dòng spring.datasource.url...databaseName=schoolmanager
   - Nếu bạn muốn dùng database khác, đổi databaseName thành tên DB của bạn.

2. Chạy file SQL để tạo bảng và dữ liệu mẫu
   - Mở SQL Server Management Studio hoặc sqlcmd.
   - Chạy file: database\schoolmanager.sql
   - File này sẽ tạo bảng [dbo].[students] và chèn dữ liệu mẫu.

3. Nếu bạn dùng database khác với tên trong file SQL
   - Thay dòng đầu tiên trong database\schoolmanager.sql thành USE [TênDatabaseCủaBạn]
   - Hoặc thay databaseName trong application.properties cho trùng với database đã tạo.

4. Lưu ý với Spring Boot
   - Hiện app đang dùng spring.jpa.hibernate.ddl-auto=none, tức là app không tự tạo bảng.
   - Nếu muốn app tự tạo bảng, đổi thành spring.jpa.hibernate.ddl-auto=update (dùng cẩn thận).

5. Chạy ứng dụng
   - mvnw.cmd spring-boot:run
   - Mở http://localhost:8080/
